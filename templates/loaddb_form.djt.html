Administrative Interface to load data

To load the database with data you must first have run the Migrate.py script to create a copy of the database ready for loading on the webserver. When you have done that you can load the data into the data base using the form below.

Load Data and Resolve references

Clicking this button will cause resolve the relationships between Shahnama data already loaded into the database. If you want to load up new data into the database be sure to tick the checkbox.

click this check box to load new Shahnama data from {{ SOURCE_DATA }} on the web server disk.

Then click this button to load the data and resolve.
